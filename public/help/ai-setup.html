AI 연결 설정

이늘 스케줄러 · CSV 자동 변환용

이 가이드는 무엇인가요?

Monday.com 같은 외부 협업 툴에서 내보낸 CSV 파일을 우리 시트 형식으로 자동 변환하기 위해 AI(LLM)를 사용합니다. 헤더 이름이 다르거나 값 형식이 달라도 AI가 한 번 분석해 매핑을 만들고, 우리 코드가 그 매핑을 모든 행에 적용합니다.

  • API 키는 관리자 본인 것을 사용합니다 (BYOK · Bring Your Own Key).
  • 키는 앱 로컬에만 저장되고 우리 서버를 거치지 않습니다.
  • AI에 전송되는 건 CSV 헤더 + 샘플 5~10행뿐입니다 (전체 행 X). 매핑이 만들어지면 나머지 변환은 모두 로컬에서.
어떤 provider를 골라야 하나? OpenAI 키 발급 Claude (Anthropic) 키 발급 Gemini (Google) 키 발급 모델 갱신 / 권장 모델 문제 해결

1어떤 provider를 골라야 하나?

OpenAI (GPT)

유료, 카드 등록 필요

가장 안정적이고 매핑 정확도가 높음. CSV 분석 1회당 비용 매우 적음 (수백~수천원 미만).

platform.openai.com ↗

Anthropic (Claude)

유료, 카드 등록 필요

긴 CSV(수천 행 헤더가 복잡)일 때 강함. JSON 출력 안정성 좋음.

console.anthropic.com ↗

Google (Gemini)

무료 한도 큼 ★

무료 한도가 가장 넓어 부담 없음. 가벼운 CSV 매핑에 충분.

aistudio.google.com ↗
처음 시작은 Google (Gemini)가 무료 한도가 커서 부담 없습니다. 본격 운영은 OpenAI나 Anthropic도 좋습니다.

2OpenAI 키 발급

  1. platform.openai.com 접속 후 로그인 (계정 없으면 가입)
  2. 우상단 프로필 → Billing에서 카드 등록 (사용한 만큼 청구, 보통 매우 소액)
  3. 좌측 메뉴 API keys → Create new secret key
  4. 키 이름 입력 (예: inel-scheduler) → Create
  5. 키가 화면에 한 번만 표시됨 → 복사
  6. 설정 → AI 연결 → Provider OpenAI 선택 → API Key 칸에 붙여넣기 → 저장
  7. 모델 갱신 버튼 → 최신 모델 5개 자동 로드 → 가장 위 모델 선택
주의: 키 노출 금지. 외부에 공유 X. 설정에서는 sk-…처럼 가려져 표시됩니다.

3Claude (Anthropic) 키 발급

  1. console.anthropic.com 접속 후 로그인 (Google 계정 가능)
  2. 좌측 Settings → Billing에서 카드 등록 + 충전 ($5부터)
  3. 좌측 API keys → Create Key
  4. 키 이름 입력 → Create Key → 키 복사
  5. 설정 → AI 연결 → Provider Anthropic (Claude) 선택 → API Key 붙여넣기 → 저장
  6. 모델 갱신 버튼 → 최신 모델 5개 로드

4Gemini (Google) 키 발급

  1. aistudio.google.com 접속 (Google 로그인)
  2. 좌측 Get API key 클릭
  3. Create API key → 새 프로젝트 자동 생성 + 키 발급 (무료 한도로 시작)
  4. 발급된 키 복사 (AIza… 형태)
  5. 설정 → AI 연결 → Provider Google (Gemini) 선택 → API Key 붙여넣기 → 저장
  6. 모델 갱신 버튼 → 최신 모델 5개 로드
무료 한도가 큰 편이라 가벼운 CSV 매핑은 비용 없이 사용 가능합니다.

5모델 갱신 / 어떤 모델을 고를까

  • 설정의 모델 갱신 버튼은 provider의 공식 모델 API를 호출해 가장 최신 5개를 다시 가져옵니다 (가장 최신 + 그 직전 버전들).
  • 새 모델이 나올 때마다 모델 갱신을 한 번 누르면 dropdown 목록이 자동으로 갱신됩니다.
  • 기본은 가장 위 모델 (= 최신)을 선택하면 됩니다.
  • 비용을 줄이고 싶으면 같은 세대의 mini / flash / haiku 같은 경량 모델을 골라도 CSV 헤더 매핑은 충분합니다.

6문제 해결

  • 모델 갱신이 실패해요 — API 키가 잘못됐거나 카드 등록이 안 된 경우. 디버그 패널의 [AI:list-models] 로그를 확인하세요.
  • CSV 분석에서 "AI 분석 실패" — 디버그 패널의 [AI:analyze] 단계별 로그 확인 → 우상단 [전체 복사]로 통째로 복사해 개발자에게 전달.
  • 매핑이 어색해요 — 다른 모델로 바꿔서 다시 분석, 또는 CSV 첫 줄 헤더가 한국어인지 확인.
  • 비용이 걱정돼요 — Google Gemini 무료 한도부터 시작하세요.
전송되는 데이터 범위: CSV 헤더 + 첫 5~10행. 전체 행은 절대 AI로 보내지 않습니다 (로컬에서 매핑 적용).
이 가이드는 앱의 [설정 → AI 연결 → 가이드 보기] 버튼으로 언제든 다시 열 수 있습니다.
