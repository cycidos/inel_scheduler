Google 계정 연결 — 빠른 시작

1

Google 계정으로 로그인

  1. 앱 우상단 톱니바퀴 → [구글 시트] 탭
  2. "Google 계정 연결" 카드의 [Google 계정으로 로그인] 클릭
  3. 시스템 브라우저에서 Google 로그인 페이지 열림 → 본인 Gmail 로 로그인
  4. 권한 동의 화면 → [허용]
  5. 앱으로 자동 복귀 → "현재 본인이메일 계정으로 로그인" 표시
2

시트 만들기 + URL 입력

  1. Google Sheets 열기 → 새 빈 스프레드시트 만들기 (이름 자유)
  2. 브라우저 주소창의 URL 전체 복사 (예: https://docs.google.com/spreadsheets/d/.../edit)
  3. 앱의 설정 → 구글 시트 탭의 "Google Sheets 링크" 입력란에 붙여넣기
  4. [연결 테스트] 클릭 → "연결 성공" 메시지 확인
시트가 비어 있어도 OK. 첫 [변경사항 저장] 시 숏폼_2026 / 롱폼_2026 / 다시보기_2026 / _settings 시트가 자동 생성됩니다.
3

(다른 PC 또는 스태프 추가 시) 시트 공유

  1. 시트 우상단 [공유] 버튼 클릭
  2. 해당 사람의 Gmail 입력 → 편집자 권한 → 보내기
  3. 그 사람은 본인 Gmail 로 로그인하면 같은 시트에 접근 가능
권한 회수는 같은 [공유] 화면에서 그 Gmail 옆 [X] 클릭.
이 문서는 앱 안에서 [구글 시트 → 설정 방법 자세히 보기] 또는 직접 URL 로 다시 열 수 있습니다.
