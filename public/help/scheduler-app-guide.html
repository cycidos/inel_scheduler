스케줄러 앱 사용 방법

이늘 스케줄러 · 관리자용 (1차 배포)

이 앱은 어떻게 쓰나요?

편집 작업 진행상황(숏폼 / 롱폼)과 자동 수집된 방송 기록(다시보기)을 한 곳에서 관리하는 협업용 스케줄러입니다. 모든 데이터는 Google Sheets와 양방향 동기화됩니다.

  • 상단 탭으로 숏폼 / 롱폼 / 다시보기 전환
  • 월별 페이지로 보기 → 같은 달 안에서 [할 일] / [완료됨] 자동 분리
  • 치지직 라이브 감지를 켜두면 다시보기 탭에 카테고리 타임라인이 자동 누적
  • 설정 패널 = [구글 시트] + [시트 설정] + [AI 연결] + [기타 설정] 네 탭
상단 바 탭 / 월별 / 할일·완료됨 필터 / 표시 행 수 / 행 잠금 시트 컬럼별 기능 행 / 열 다루기 다시보기 자동 누적 시트 설정 ↔ 시트 기능 연계 되살리기 / 단축키 동기화 흐름

2탭 / 월별 보기 / 할 일·완료됨 분리

  • 탭: 숏폼 / 롱폼 / 다시보기 — 각 탭은 독립된 시트 (예: 숏폼_2026)와 매핑
  • 월별 페이지: 화살표 < 2026.04 >로 같은 시트 안에서 월 단위로 묶어 봄. 데이터 없는 월은 화살표 비활성화.
  • 할 일 / 완료됨 분리 (숏폼·롱폼만): 업로드 컬럼이 완이면 [완료됨] 그룹, 아니면 [할 일] 그룹. 그룹 헤더의 ▾를 클릭해 접거나 펼칠 수 있어요. 접힘 상태는 탭별로 저장됩니다.
다시보기 탭은 그룹 분리 없이 단일 목록으로 표시됩니다.

3필터 / 표시 행 수 / 행 잠금

① 할일만 보기 / 모두 보기 토글

월 페이저 옆에 위치. 기본은 [할일만 보기].

  • 할일만 보기: [할 일] 그룹만 표시, [완료됨] 그룹 숨김.
  • 모두 보기: [할 일] + [완료됨] 모두 표시.
탭별로 따로 저장됩니다.

② 그룹별 표시 행 수 (기본 10, 최대 30)

그룹 헤더 우측에 표시 10 / 23 · 13개 더처럼 현재 표시중/총 개수가 보이고, − / + 버튼으로 표시 행 수를 5단위로 조절합니다.

  • 기본값 10개, 최소 5개, 최대 30개까지 늘릴 수 있어요.
  • 표시 수를 넘는 행은 그룹 마지막에 현재 N개 표시 중 · M개 숨김 안내 줄이 나타납니다. 숨겨진 항목을 보려면 헤더의 +를 눌러 표시 수를 더 늘리세요.
  • 그룹 (할 일 / 완료됨)별로 따로, 그리고 탭별로 따로 저장됩니다.

③ 행 잠금 — 업로드 = 완 처리

업로드 토글 셀을 완으로 체크하면:

  • 해당 행 전체가 회색으로 표시되어 끝난 작업임이 한눈에 보입니다.
  • 업로드 셀을 제외한 모든 셀이 편집 잠금 상태가 되어 실수로 값이 바뀌지 않습니다 (드롭다운, 날짜 선택, 직접 입력 모두 비활성).
  • 그 행은 [완료됨] 그룹으로 자동 이동합니다 (모두 보기 모드에서 확인 가능).
  • 다시 편집해야 하면 업로드 셀을 한 번 더 클릭해서 완을 해제하면 잠금이 풀립니다.
행 자체의 순서 이동 / 삭제는 잠금 상태에서도 좌측 ☰ / − 버튼으로 가능합니다 (관리자가 정리할 수 있도록).

4시트 컬럼별 기능

각 컬럼은 타입을 가지고 있고, 타입에 따라 클릭 시 다른 편집기가 열립니다. shared = 영상 1편당 1번 (썸네일러/영상편집자 두 행이 합쳐진 셀), 그 외는 역할별 (썸네일러 / 영상편집자 각각 입력).

업로드 status shared

유튜브 업로드 완료 표시. 클릭 시 완 ↔ 빈값 토글. 이 값이 할 일 / 완료됨 그룹 분리 기준.

방송일 date shared

월별 페이지 분류 기준. 클릭 시 달력 입력기.

영상제목 text shared

영상 제목 자유 입력.

영상 카테고리 select shared

+ 클릭 → 카테고리 검색창. 시드(207개)+사용자 추가분에서 자동완성. 없으면 치지직 비공식 검색으로 폴백, 클릭 시 자동 등록됩니다. 직접 입력 후 Enter도 가능. 다중 태그.

↻ 다시보기 자동 감지에서도 새 카테고리 자동 등록
담당자 select

설정의 편집자/썸네일러 등록에 추가된 인원이 dropdown 후보로 떠요. 행마다 두 줄(썸네일러/영상편집자)에 각각 할당.

⇆ 설정 → 시트 설정 → 편집자/썸네일러 등록
편집 유형 preset

옵션 dropdown (미설정 / 하이라이트 편집 / 컷편집 / 무편집 / 풀편집 / -) + 직접 입력 가능. 영상편집자 행 기본 미설정, 썸네일러 행 기본 -.

자막 preset

옵션 dropdown (미설정 / 기본 자막 / 자막X / 효과자막 포함 / -) + 직접 입력 가능. 영상편집자 행 위주 입력.

작업시작일 / 납품일 date

담당자별 시작/마감 일정. 클릭 시 달력 입력기.

작업상태 status

색 칩 dropdown — 할 일 / 진행 중 / 완료 그룹으로 정리되어 표시. 후보 값은 설정에서 자유 추가/삭제 가능.

⇆ 설정 → 시트 설정 → 작업상태 옵션
원본 공유 / 납품 공유 url

URL 입력. 입력 후 링크 버튼으로 보기 모드(클릭 가능 링크) 전환, 수정 버튼으로 다시 편집 모드.

카테고리 타임라인 (다시보기) text shared

방송 감지 ON 시 카테고리가 바뀔 때마다 00:35:21 - 발로란트 형식으로 한 셀에 멀티라인 누적.

5행 / 열 다루기

행

  • 행 위에 마우스 올리면 왼쪽 36px 영역에 ☰ 드래그 핸들 + − 삭제 버튼 표시
  • 드래그 핸들로 행 순서 변경 (실시간 swap)
  • 표 하단 + 새 행 추가 버튼으로 빈 행 추가 → 컬럼 타입에 맞춘 기본값 자동 채움

열 (헤더)

  • 헤더 hover → 좌측 ⋮⋮ 핸들로 열 순서 변경, −로 삭제
  • 헤더 라벨 클릭으로 라벨 직접 편집, 좌측 타입 아이콘 클릭으로 컬럼 타입 변경
  • 헤더 우측 모서리 드래그로 컬럼 너비 리사이즈
상단의 열 추가 버튼은 1차 배포에서 숨겨져 있습니다. 헤더 편집은 가능합니다.

6다시보기 자동 누적

방송감지 토글이 ON일 때 동작:

  • 치지직 폴링으로 방송 시작이 감지되면 다시보기 탭 마지막에 새 행 자동 추가 (방송일=오늘, 영상제목=현재 방송 제목)
  • 방송 중 카테고리가 바뀔 때마다 그 행의 카테고리 타임라인 셀에 방송 업타임 - 카테고리 한 줄 누적
  • 새로 발견된 카테고리는 사용자 카테고리에 자동 등록되어 다음부터 영상 카테고리 검색에서도 잡힘
감지 주기는 설정 → 시트 설정 → 감지 주기에서 0.5초 ~ 30초 사이로 조정.

7시트 설정 ↔ 시트 기능 연계 (가장 중요)

설정 패널에서 등록·조정한 값이 시트의 어디에 어떻게 반영되는지 정리했습니다.

① 편집자 / 썸네일러 등록 → 담당자 셀 후보 + 스태프 인스톨러

설정 → 시트 설정 → 편집자/썸네일러 등록 역할(파랑/주황 라디오) + 이름 + Gmail 입력 → [추가].
→
시트 → 담당자 컬럼 셀 / 스태프 빌드 담당자 셀 dropdown 후보 (역할별 필터링). [기타 설정] 에서 해당 인원의 스태프 .exe 인스톨러 빌드 가능.

새 스태프 합류 시: 설정에서 등록 → 시트 [공유] 에 Gmail 추가 → 인스톨러 빌드 → 전달.

② 작업상태 옵션 → 작업상태 셀 dropdown

설정 → 시트 설정 → 작업상태 옵션 기본: Wait / Wip / Done / Publish / Retake / Omit. 자유롭게 추가·삭제 가능.
→
시트 → 작업상태 컬럼 셀 색 칩 dropdown 후보. 할 일 / 진행 중 / 완료 그룹으로 묶어 표시.

③ 치지직 방송 링크 + 감지 주기 → 다시보기 자동 누적

설정 → 시트 설정 → 치지직 방송 링크 / 감지 주기 치지직 채널 URL 등록 + polling 간격 (0.5초~30초).
→
시트 → 다시보기 탭 자동 행 + 카테고리 타임라인 상단 [방송감지 On] 켜면 채널의 라이브 상태/제목/카테고리를 polling. 카테고리가 바뀔 때마다 다시보기 시트에 누적.

④ 영상 카테고리 시드 / 사용자 카테고리 → 카테고리 검색 dropdown

앱 내장 시드 (~207개) + 사용자 추가분 치지직 라이브 크롤링 결과 + 폴백 검색에서 자동 등록된 카테고리.
→
시트 → 영상 카테고리 + 검색 + 클릭 → 검색 → 시드/사용자분 자동완성. 없으면 치지직 라이브에서 실시간 폴백 검색 → 클릭 시 자동 등록.

⑤ 되살리기 스택 크기 → Ctrl+Z 가능 단계

설정 → 시트 설정 → 되살리기 / 다시실행 설정 슬라이더로 5~50단계 조정 (기본 10).
→
시트 전반 → Ctrl+Z / Ctrl+Shift+Z 같은 셀 연속 입력은 한 단계로 묶어 처리.

9동기화 흐름

  • 앱 시작 시 자동 다운로드 (Google 계정 로그인 + 시트 URL 모두 등록되어 있으면)
  • 일정 새로고침: 3개 탭(숏폼/롱폼/다시보기) 순차 import. 진행률은 메타 영역의 핑크 막대로 표시.
  • 변경사항 저장: 3개 탭 순차 export. 시트가 없으면 자동 생성 (시트 이름 규칙: 숏폼_2026 / 롱폼_2026 / 다시보기_2026).
  • 설정 자동 동기화: _settings 시트도 함께 읽어 편집자 명단 / 컬럼 구성 / 정렬 / 작업상태 옵션을 복원.
초기 셋업은 설정 → 구글 시트 탭에서 [Google 계정으로 로그인] + Sheet URL 입력.
