Chzzk 스트리머와 영상 편집 협업자를 위한 데스크톱 스케줄러

이 앱은 무엇인가요?

개인 방송인이 외주 영상 편집자 / 썸네일러와 함께 작업할 때, 영상 제작 진행 상황과 방송 기록을 한 곳에서 관리할 수 있도록 만든 Windows 데스크톱 앱입니다.

모든 작업 데이터는 사용자 본인이 소유한 Google Sheet 한 장과 양방향으로 동기화됩니다. 앱이 별도의 서버를 거치지 않으며, 사용자 본인의 Google 계정 권한으로 본인의 시트만 접근합니다.

주요 기능

일정 관리 숏폼 / 롱폼 / 다시보기 탭별 작업 일정 관리. 월별 페이지 + 할 일 / 완료됨 분리.
편집자 협업 편집자 / 썸네일러별 본인 권한 인스톨러 발급. 본인 담당 작업만 표시.
방송 자동 기록 Chzzk 라이브 감지 → 다시보기 탭에 카테고리 / 제목 타임라인 자동 누적.
Google Sheet 동기화 사용자가 소유한 Google Sheet 와 양방향 동기화. 모든 작업 데이터의 단일 원본.

다운로드 / 사용 안내

현재 비공개 베타 단계로 소수의 개별 사용자에게만 배포되고 있습니다. 사용을 원하시면 아래 이메일로 문의해 주세요.

Windows 10 / 11 64-bit 환경에서 동작합니다.

OAuth 권한 안내 (Google 계정 로그인)

앱이 사용자에게 요청하는 Google OAuth 권한은 다음과 같습니다. 모든 권한은 사용자 본인이 소유한 시트 의 범위 안에서만 사용됩니다.

  • Google Sheets 접근 (spreadsheets) — 사용자가 입력한 시트 URL 의 일정 데이터를 앱과 양방향 동기화합니다. 앱이 자기 데이터베이스를 따로 두지 않고 사용자의 시트를 그대로 저장소로 사용합니다.
  • Google Drive (앱이 만든 파일 한정) (drive.file) — 시트 생성 / 메타데이터 조회 등 시트가 작동하는 데 필요한 최소 범위로만 사용합니다.
  • 이메일 조회 (userinfo.email) — 로그인한 본인 계정 식별 + 편집자 빌드에서 임베드된 이메일과 일치하는지 확인용으로만 사용합니다.

앱 외부로 데이터를 전송하지 않으며, 개발자 서버 / 통계 / 트래킹 / 광고는 사용하지 않습니다. 자세한 내용은 개인정보처리방침 을 참고하세요.
